## 📊 Architecture

```
┌─────────────────────────────────────────┐
│       Protected Spine (Kernel)          │
│      Immutable Mission & Rules          │
└─────────────────┬───────────────────────┘
                  │
        ┌─────────┴─────────┐
        │                   │
┌───────▼────────┐  ┌──────▼────────┐
│  Brick System  │  │  VERA Gate    │
│   Isolation    │  │ Verification  │
└───────┬────────┘  └──────┬────────┘
        │                  │
        └────────┬─────────┘
                 │
       ┌─────────▼──────────┐
       │  Append-Only       │
       │     Ledger         │
       └─────────┬──────────┘
                 │
       ┌─────────┴──────────────────┐
       │                            │
┌──────▼──────┐           ┌─────────▼────────┐
│ GHOST NODE  │           │  TRUTH NODE      │
│ (Classified)│           │  (Intel Verify)  │
└─────────────┘           └──────────────────┘

                 ┌─────────▼──────────┐
                 │  PHOENIX NODE      │
                 │  (Disaster Only)   │
                 └────────────────────┘
```

### Core Components

- **Protected Spine**: Immutable mission directives (never corrupted)
- **Brick Isolation**: Decomposed, isolated task units with failure containment
- **Braided Routing**: Work path + contradiction path for verification
- **VERA Gate**: Mandatory pre-commit verification (blocks unsafe operations)
- **Append-Only Ledger**: Complete audit trail (no silent mutations)
- **Cold-Stitch Recovery**: Deterministic rollback and rebuild from trusted spine
